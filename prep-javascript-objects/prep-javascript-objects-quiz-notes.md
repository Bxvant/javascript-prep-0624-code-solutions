# prep-javascript-objects-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What data types can object properties hold?
  they can hold strings, numbers, booleans, etc..
- Describe the syntax (structure) of object-literals in JavaScript.
  object literals begin with the variable declaration, followed by the assignment operator. Following the assignment operator you can begin with the open curly braces, in which you would type your keys and their values, with each key and its corresponding value being separated from other keys and their values with a comma and closing the object literal with the closing curly brace.
